# Neccessary command line
- typescript
```
npm init -y
npm i typescript
tsc --init
```
- redis
```
redis-server // start redis server
redis-cli
redis-commander // run redis gui
redis-cli shutdown
```
- tests (jest)
```
jest // run all tests
```
